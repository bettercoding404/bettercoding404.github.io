## 基础概念
直角三角形图案是由字符组成的，呈现出直角三角形形状的图形。在Python中，我们通过使用循环（如`for`循环和`while`循环）以及条件语句（如`if`语句）来控制字符的打印位置和数量，从而形成直角三角形。

## 使用方法

### 使用`for`循环打印直角三角形
```python
# 打印直角三角形
size = 5
for i in range(1, size + 1):
    for j in range(i):
        print("*", end="")
    print()
```
在这段代码中：
1. 外层`for`循环控制行数，`range(1, size + 1)`表示从1到`size`（这里`size`为5），共5行。
2. 内层`for`循环控制每行打印的星号数量，`range(i)`表示第`i`行打印`i`个星号。
3. `print("*", end="")`表示打印星号且不换行，`print()`表示换行，以便开始新的一行。

### 使用`while`循环打印直角三角形
```python
# 使用while循环打印直角三角形
size = 5
i = 1
while i <= size:
    j = 1
    while j <= i:
        print("*", end="")
        j += 1
    print()
    i += 1
```
这段代码使用`while`循环实现了与`for`循环相同的功能：
1. 外层`while`循环通过变量`i`控制行数，只要`i`小于等于`size`，就继续循环。
2. 内层`while`循环通过变量`j`控制每行打印的星号数量，只要`j`小于等于`i`，就打印星号并将`j`加1。
3. 内层循环结束后，通过`print()`换行，然后将`i`加1，开始新的一行。

## 常见实践

### 打印不同方向的直角三角形
#### 左对齐直角三角形
```python
# 打印左对齐直角三角形
size = 5
for i in range(1, size + 1):
    for j in range(size - i):
        print(" ", end="")
    for k in range(i):
        print("*", end="")
    print()
```
在这段代码中，第一个内层`for`循环打印空格，用于将星号左对齐，第二个内层`for`循环打印星号。

#### 倒置直角三角形
```python
# 打印倒置直角三角形
size = 5
for i in range(size, 0, -1):
    for j in range(i):
        print("*", end="")
    print()
```
这里外层`for`循环从`size`递减到1，从而实现倒置的直角三角形。

### 打印带特定字符的直角三角形
```python
# 打印带特定字符的直角三角形
size = 5
char = '@'
for i in range(1, size + 1):
    for j in range(i):
        print(char, end="")
    print()
```
通过修改`char`变量的值，可以打印出由不同字符组成的直角三角形。
