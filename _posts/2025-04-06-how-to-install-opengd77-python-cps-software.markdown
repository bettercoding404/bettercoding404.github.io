## 基础概念
### OpenGD77
OpenGD77 是一款具有多种功能的无线电设备，支持数字和模拟模式，在业余无线电、商业通信等领域有广泛应用。

### Python CPS 软件
CPS 即 “Configuration Programming Software”（配置编程软件）。基于 Python 的 OpenGD77 CPS 软件利用 Python 的灵活性和丰富的库资源，允许用户对 OpenGD77 进行深度配置，如设置频率、通信模式、用户信息等。

## 安装步骤
### 环境准备
确保你的系统已经安装了 Python 环境，推荐使用 Python 3.6 及以上版本。你可以从[Python 官方网站](https://www.python.org/downloads/)下载适合你操作系统的安装包进行安装。

### 安装依赖库
在安装 OpenGD77 Python CPS 软件之前，需要安装一些必要的依赖库。打开命令行工具（在 Windows 上是命令提示符或 PowerShell，在 Linux 和 macOS 上是终端），输入以下命令安装 `pyserial` 和 `wxPython` 库：
```bash
pip install pyserial wxPython
```
### 下载 OpenGD77 Python CPS 软件
你可以从项目的官方 GitHub 仓库下载软件代码。在命令行中输入以下命令克隆仓库：
```bash
git clone https://github.com/OpenGD77/OpenGD77-CPS
```
或者直接从 GitHub 页面上下载 ZIP 压缩包，解压到你希望的目录。

### 运行安装脚本
进入下载的 `OpenGD77-CPS` 目录，在命令行中输入：
```bash
python setup.py install
```
这将安装 OpenGD77 Python CPS 软件到你的系统中。

## 使用方法
### 连接设备
使用合适的数据线将 OpenGD77 无线电设备连接到计算机。确保设备已开启并处于正确的模式。

### 启动软件
在命令行中输入 `opengd77_cps` 来启动软件。或者在安装目录中找到可执行文件并双击运行。

### 配置设备
软件启动后，你将看到一个图形化界面。通过界面上的各个选项卡和按钮，你可以进行以下操作：
- **频率设置**：在 “Frequency” 选项卡中，输入你需要的发射和接收频率。
- **通信模式选择**：在 “Mode” 选项卡中，选择数字或模拟模式，并配置相关参数。
- **用户信息设置**：在 “User” 选项卡中，填写用户姓名、呼号等信息。

### 保存和写入配置
完成所有设置后，点击 “Save” 按钮保存配置文件，然后点击 “Write to Radio” 按钮将配置写入到 OpenGD77 设备中。

## 常见实践
### 频率规划
在业余无线电通信中，合理的频率规划至关重要。通过 OpenGD77 Python CPS 软件，你可以根据当地的频段分配规则和实际需求，设置多个频率组，方便在不同场景下快速切换。
```python
# 示例代码：设置频率组

    {"tx": 438.500, "rx": 438.500},
    {"tx": 439.000, "rx": 439.000}
]
```

### 模式切换
根据通信对象和环境，灵活切换数字和模拟模式。例如，在与传统模拟设备通信时，切换到模拟模式；在需要更高语音质量和数据传输时，使用数字模式。

### 备份和恢复配置
定期备份设备配置是一个好习惯。在软件中，你可以通过 “Backup” 功能将当前配置保存为文件。当设备出现问题或需要重新配置时，使用 “Restore” 功能加载备份文件。
