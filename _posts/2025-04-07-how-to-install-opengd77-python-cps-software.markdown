## 基础概念
OpenGD77 Python CPS 软件基于 Python 语言开发，旨在实现对 GD77 对讲机的编程和配置。CPS 即“Configuration Programming Software”（配置编程软件）的缩写，通过该软件，用户可以方便地设置对讲机的各种参数，如频率、信道、发射功率等。它不仅提高了对讲机配置的效率，还增强了配置的准确性和灵活性。

## 安装步骤

### 准备工作
1. **安装 Python**：确保系统中已经安装了 Python。建议安装 Python 3.6 及以上版本。可以从 Python 官方网站（https://www.python.org/downloads/）下载适合您操作系统的安装包进行安装。
2. **安装依赖库**：OpenGD77 Python CPS 软件依赖一些 Python 库，主要包括 `pyserial` 和 `pyusb`。可以使用 `pip` 工具进行安装。打开命令行终端，输入以下命令：
```bash
pip install pyserial pyusb
```

### 下载 OpenGD77 Python CPS 软件
可以从开源代码托管平台如 GitHub 上获取 OpenGD77 Python CPS 软件的源代码。通过以下步骤进行下载：
1. 打开浏览器，访问项目的 GitHub 页面（例如：https://github.com/your-repo-url）。
2. 点击页面上的 “Code” 按钮，选择 “Download ZIP” 下载压缩包。
3. 将下载的压缩包解压到您希望安装的目录。

### 配置软件
进入解压后的软件目录，找到配置文件（通常是 `config.ini` 之类的文件名）。根据您的实际需求，修改配置文件中的参数，如串口设置、对讲机型号等。例如：
```ini
[general]
port = COM3  # 根据实际连接的串口进行修改
baudrate = 9600
```

### 安装完成
完成上述步骤后，OpenGD77 Python CPS 软件就安装成功了。您可以通过运行主程序文件（通常是 `.py` 后缀的文件）来启动软件。

## 使用方法
1. **启动软件**：在命令行终端中，进入软件目录，运行主程序文件。例如，如果主程序文件名为 `opengd77_cps.py`，则输入以下命令：
```bash

```
2. **连接对讲机**：将 GD77 对讲机通过合适的数据线连接到计算机，并确保在配置文件中设置的串口正确无误。启动软件后，软件会尝试自动检测并连接对讲机。
3. **配置参数**：软件启动后，会显示一个用户界面（可能因版本而异），通过该界面可以直观地设置对讲机的各种参数。例如，在频率设置界面，可以输入所需的发射和接收频率；在信道设置界面，可以添加、删除或修改信道参数。
4. **保存设置**：完成参数配置后，点击保存按钮（通常是一个磁盘图标），软件会将配置信息写入对讲机。
