## 基础概念
数据结构是计算机存储、组织数据的方式。在 Python 中，常见的数据结构分为可变（mutable）和不可变（immutable）两种类型。可变数据结构可以在创建后修改其内容，例如列表和字典；不可变数据结构一旦创建，其内容不能被修改，例如元组和集合（集合中的元素必须是不可变类型）。初始化数据结构就是在程序中创建这些数据结构的实例，并赋予它们初始值。

## 使用方法

### 列表初始化
列表是 Python 中最常用的数据结构之一，它可以包含不同类型的元素，并且是可变的。
- **空列表初始化**：
```python
my_list = []
```
- **初始化带有元素的列表**：
```python
my_list = [1, 2, 3, 'hello']
```
- **使用列表推导式初始化列表**：
```python
# 创建一个包含 1 到 10 的平方的列表
squares = [i**2 for i in range(1, 11)]
```

### 字典初始化
字典是一种无序的数据结构，它以键值对（key - value pairs）的形式存储数据，便于快速查找。
- **空字典初始化**：
```python
my_dict = {}
```
- **初始化带有键值对的字典**：
```python
my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}
```
- **使用字典推导式初始化字典**：
```python
# 创建一个键为 1 到 5，值为键的平方的字典
squares_dict = {i: i**2 for i in range(1, 6)}
```

### 集合初始化
集合是一个无序且唯一的数据结构，其中的元素不能重复。
- **空集合初始化**：
```python
my_set = set()
```
注意不能使用 `{}` 来初始化空集合，因为 `{}` 会创建一个空字典。
- **初始化带有元素的集合**：
```python
my_set = {1, 2, 3, 4}
```
- **使用集合推导式初始化集合**：
```python
# 创建一个包含 1 到 10 中所有偶数的集合
even_set = {i for i in range(1, 11) if i % 2 == 0}
```

### 元组初始化
元组是不可变的数据结构，通常用于存储一组相关的数据。
- **空元组初始化**：
```python
my_tuple = ()
```
- **初始化带有元素的元组**：
```python
my_tuple = (1, 2, 3, 'world')
```
注意，当元组只有一个元素时，需要在元素后面加上逗号，例如 `(1,)`，以区别于普通的表达式。

## 常见实践

### 根据需求选择合适的数据结构
在初始化数据结构之前，需要根据程序的需求来选择合适的数据结构。例如，如果需要频繁地插入和删除元素，列表可能是一个不错的选择；如果需要快速查找，字典或集合可能更合适；如果数据不需要修改且顺序不重要，元组或集合可能是最佳选择。

### 初始化时设置默认值
在某些情况下，初始化数据结构时设置默认值可以简化代码逻辑。例如，在字典中可以使用 `defaultdict` 来设置默认值：
```python
from collections import defaultdict

# 创建一个默认值为 0 的字典
my_dict = defaultdict(int)
print(my_dict['key'])  # 输出 0，即使 'key' 不存在
```

## 最佳实践

### 性能优化
- **使用生成器表达式代替列表推导式（在需要惰性求值时）**：生成器表达式不会一次性生成所有元素，而是在需要时逐个生成，节省内存。
```python
# 生成器表达式
gen = (i**2 for i in range(1, 1000000))
# 列表推导式会一次性生成所有元素，占用大量内存
# my_list = [i**2 for i in range(1, 1000000)]
```
- **对于大型数据集，优先使用 `numpy` 数组而不是普通列表**：`numpy` 数组在数值计算上更加高效，并且占用更少的内存。

### 代码可读性优化
- **使用描述性的变量名**：为初始化的数据结构使用清晰、有意义的变量名，使代码更易于理解。
- **将复杂的初始化逻辑封装成函数**：如果初始化过程涉及复杂的计算或条件判断，将其封装成函数可以提高代码的模块化和可维护性。
