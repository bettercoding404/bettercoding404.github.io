Python Cheat Sheet PDF是一个包含了Python编程语言关键信息的文档，它涵盖了从基础语法到高级特性的各个方面。

```python

age = 30

# 数据类型
my_list = [1, 2, 3, "four"]
my_dict = {"name": "Alice", "age": 25}

# 控制流语句

    print("Adult")
else:
    print("Minor")

for num in my_list:
    print(num)

    print(count)
    count += 1
```
