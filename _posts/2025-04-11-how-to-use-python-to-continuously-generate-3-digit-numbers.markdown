## 基础概念
### 什么是三位数
三位数是指从100到999之间的整数。在Python中，我们可以通过特定的语法和函数来生成这个范围内的数字。

### 连续生成的含义
连续生成意味着按照一定的顺序或规律，逐个产生三位数。这可以是顺序递增（如100, 101, 102...），也可以是随机但持续不断地生成。

## 使用方法
### 使用`range`函数
`range`函数在Python中用于生成一个整数序列。要生成三位数，可以使用以下代码：

```python
for num in range(100, 1000):
    print(num)
```

在这段代码中，`range(100, 1000)`生成了一个从100（包含）到1000（不包含）的整数序列。通过`for`循环，我们可以逐个打印出这些三位数。

### 使用`random`模块生成随机三位数
`random`模块可以用于生成随机数。以下是生成随机三位数的代码：

```python
import random

for _ in range(10):  # 生成10个随机三位数
    random_num = random.randint(100, 999)
    print(random_num)
```

在这段代码中，`random.randint(100, 999)`会在每次调用时生成一个介于100到999之间（包含100和999）的随机整数。`for _ in range(10)`表示循环10次，因此会生成10个不同的随机三位数。

## 常见实践
### 用于数据测试
在开发软件或进行算法测试时，我们经常需要使用一些样本数据。生成三位数可以作为简单的数据样本。例如，我们可以生成一系列三位数来测试排序算法：

```python
import random

# 生成100个随机三位数作为测试数据
test_data = [random.randint(100, 999) for _ in range(100)]

# 对测试数据进行排序
sorted_data = sorted(test_data)
print(sorted_data)
```

### 密码生成（简单示例）
虽然简单的三位数作为密码并不安全，但在某些简单场景下可以作为示例。以下是生成随机三位数作为“密码”的代码：

```python
import random

password = random.randint(100, 999)
print(f"生成的简单密码: {password}")
```

## 最佳实践
### 性能优化
如果需要生成大量的三位数，使用生成器表达式可以提高性能。例如：

```python
def three_digit_generator():
    for num in range(100, 1000):
        yield num

gen = three_digit_generator()
for _ in range(5):  # 获取前5个三位数
    print(next(gen))
```

生成器表达式不会一次性将所有数字存储在内存中，而是按需生成，这对于处理大量数据时非常高效。

### 代码可读性优化
为了使代码更易读，可以将生成三位数的逻辑封装成函数。例如：

```python
import random


def generate_random_three_digit():
    return random.randint(100, 999)


for _ in range(5):
    print(generate_random_three_digit())
```

这样，当代码逻辑更复杂时，通过函数封装可以使代码结构更清晰。
