## 简介

## 基础概念
打印格式化的核心目标是控制输出的外观和布局。在 Python 中，这主要涉及如何将变量的值嵌入到字符串中，并对其进行格式化，例如设置数字的精度、对齐文本、添加填充字符等。

## 使用方法
### 旧风格格式化（% 操作符）
这是 Python 中较老的格式化方式，使用 `%` 操作符来指定格式。

```python
name = "Alice"
age = 30
print("My name is %s and I'm %d years old." % (name, age))
```

在这个例子中，`%s` 是字符串占位符，`%d` 是整数占位符。`%` 操作符后面的元组包含要插入的值。

### 新风格格式化（str.format() 方法）
从 Python 2.6 开始引入的新风格格式化，使用 `str.format()` 方法。

```python
name = "Bob"
age = 25
print("My name is {} and I'm {} years old.".format(name, age))
```

这里使用花括号 `{}` 作为占位符，`format()` 方法中的参数按顺序替换占位符。也可以通过索引指定顺序：

```python
print("My name is {1} and I'm {0} years old.".format(age, name))
```

### f - 字符串格式化（Python 3.6+）
这是 Python 3.6 引入的一种简洁直观的格式化方式。

```python
name = "Charlie"
age = 35
print(f"My name is {name} and I'm {age} years old.")
```

在 f - 字符串中，可以直接在花括号内嵌入变量或表达式，并且支持更复杂的格式化选项。

```python
pi = 3.1415926
print(f"Pi rounded to 3 decimal places is {pi:.3f}")
```

## 常见实践
### 格式化数字
- **设置精度**：

```python
num = 123.456789
print(f"{num:.2f}")  # 保留两位小数
```

- **添加千位分隔符**：

```python
large_num = 1234567890
print(f"{large_num:,}")  # 添加千位分隔符
```

### 格式化字符串
- **限制字符串长度**：

```python
long_str = "This is a very long string"
print(f"{long_str:.10}")  # 截取前 10 个字符
```

### 对齐文本
- **左对齐**：

```python
text = "Hello"
print(f"{text:<10}")  # 左对齐，宽度为 10
```

- **右对齐**：

```python
print(f"{text:>10}")  # 右对齐，宽度为 10
```

- **居中对齐**：

```python
print(f"{text:^10}")  # 居中对齐，宽度为 10
```

### 处理日期和时间
```python
import datetime

now = datetime.datetime.now()
print(f"{now:%Y-%m-%d %H:%M:%S}")
```

## 最佳实践
### 可读性优先
选择最能清晰表达意图的格式化方式。f - 字符串通常在简洁性和可读性方面表现出色，但对于复杂的格式化逻辑，`str.format()` 可能更具优势。

### 灵活性与扩展性
考虑代码的未来变化，选择具有灵活性的格式化方法。例如，如果需要动态更改格式选项，`str.format()` 可能更合适，因为它允许在运行时调整格式。

### 避免复杂的格式化逻辑
尽量将复杂的格式化逻辑提取到单独的函数或方法中，以提高代码的可维护性。

## 小结
Python 提供了多种打印格式化的方法，每种方法都有其特点和适用场景。旧风格格式化简洁但功能有限，新风格格式化更加灵活，而 f - 字符串格式化则简洁直观，适用于现代 Python 代码。通过掌握这些方法和最佳实践，开发者能够更加高效地控制输出的格式，提升代码的可读性和可维护性。
