## 基础概念
在 Python 里，百分号（%）主要有两个核心用途：一是作为数学运算符中的取模运算符，二是用于字符串格式化。取模运算返回除法的余数，而字符串格式化则允许我们以特定的方式将变量值嵌入到字符串中。

## 作为取模运算符
### 使用方法
取模运算符（%）用于计算两个数相除的余数。其语法为 `number1 % number2`，其中 `number1` 是被除数，`number2` 是除数，运算结果是相除后的余数。

### 代码示例
```python
# 整数取模
result1 = 10 % 3
print(result1)  # 输出 1，因为 10 除以 3 商为 3，余数为 1

# 浮点数取模
result2 = 10.5 % 2.5
print(result2)  # 输出 0.5，10.5 除以 2.5 商为 4，余数为 0.5
```

## 字符串格式化
### 使用方法
在字符串格式化中，百分号用于指定嵌入变量的位置和格式。语法形式为：`"格式化字符串" % (变量 1, 变量 2,...)`。格式化字符串中包含以 `%` 开头的格式说明符，如 `%d` 用于整数，`%f` 用于浮点数，`%s` 用于字符串等。

### 代码示例
```python
name = "Alice"
age = 25
message = "My name is %s and I am %d years old." % (name, age)
print(message)  # 输出 "My name is Alice and I am 25 years old."

# 格式化浮点数
pi = 3.1415926
formatted_pi = "Pi is approximately %.2f" % pi
print(formatted_pi)  # 输出 "Pi is approximately 3.14"，保留两位小数
```

## 常见实践
### 数学计算场景
在数学计算中，取模运算符常用于判断一个数是否能被另一个数整除，或者进行循环计数等操作。例如，在循环中每 `n` 次执行特定操作：
```python
for i in range(10):
    if i % 3 == 0:
        print(f"{i} 是 3 的倍数")
```

### 字符串处理场景
在字符串处理方面，字符串格式化使用百分号可以方便地将变量值嵌入到文本中，生成动态的消息或报告。例如，生成文件路径：
```python
folder = "data"
file_name = "report.txt"
file_path = "/%s/%s" % (folder, file_name)
print(file_path)  # 输出 "/data/report.txt"
```

## 最佳实践
### 选择合适的格式化方式
虽然百分号格式化字符串是一种经典方法，但 Python 还提供了更现代的格式化方式，如 `str.format()` 方法和 f-strings（Python 3.6+）。在新代码中，推荐优先使用 f-strings，因为它们语法更简洁、易读。例如：
```python
name = "Bob"
age = 30
# 使用 f-strings
new_message = f"My name is {name} and I am {age} years old."
print(new_message)
```

### 注意精度和类型匹配
在进行字符串格式化时，要确保格式说明符与变量类型相匹配。例如，使用 `%d` 格式化浮点数会导致错误。同时，对于浮点数格式化，要注意设置合适的精度，避免数据丢失或不精确的显示。

## 小结
Python 中的百分号（%）具有取模运算和字符串格式化两个主要功能。取模运算用于计算余数，在数学和循环逻辑中有广泛应用；字符串格式化则帮助我们将变量值以特定格式嵌入到字符串中。在实际编程中，要根据具体场景选择合适的用法，并注意遵循最佳实践，以提高代码的可读性和正确性。
