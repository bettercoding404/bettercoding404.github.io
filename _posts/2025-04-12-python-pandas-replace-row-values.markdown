## 基础概念
Pandas 是 Python 中用于数据处理和分析的核心库之一。数据框（DataFrame）是 Pandas 中最常用的数据结构，它类似于电子表格或数据库表，由行和列组成。替换行值就是在数据框中，针对特定的行，用新的值替换掉原来的值。这一操作可以基于各种条件进行，例如值的匹配、正则表达式匹配等。

## 使用方法

### 简单替换
简单替换是指直接将某一行中的特定值替换为新的值。假设我们有一个包含学生成绩的数据框：

```python
import pandas as pd

# 创建数据框
data = {
    '姓名': ['小明', '小红', '小李'],
    '数学': [85, 90, 78],
    '英语': [70, 82, 88]
}
df = pd.DataFrame(data)

# 简单替换：将小李的数学成绩从 78 替换为 80
df.loc[df['姓名'] == '小李','数学'] = 80
print(df)
```

### 基于条件替换
我们可以根据某些条件来替换行值。例如，将所有英语成绩小于 80 分的替换为 80 分：

```python
df.loc[df['英语'] < 80, '英语'] = 80
print(df)
```

### 正则表达式替换
当需要进行更复杂的文本匹配和替换时，可以使用正则表达式。假设我们的数据框中有一个包含城市名称的列，并且名称中包含一些错误的格式，我们可以使用正则表达式来修正：

```python
data = {
    '城市': ['北京,China', '上海;China', '广州,China']
}
df_city = pd.DataFrame(data)

# 使用正则表达式将逗号和分号替换为空格
df_city['城市'] = df_city['城市'].replace(r'[;,]',' ', regex=True)
print(df_city)
```

## 常见实践

### 数据清洗中的行值替换
在数据清洗过程中，经常会遇到一些无效值或错误值，需要进行替换。例如，数据中可能存在用 `?` 表示的缺失值，我们可以将其替换为更合适的值：

```python
data_with_missing = {
    '数值': [10, '?', 20, '?', 30]
}
df_missing = pd.DataFrame(data_with_missing)

# 将 '?' 替换为 NaN，以便后续处理缺失值
df_missing['数值'] = df_missing['数值'].replace('?', pd.NA)
print(df_missing)
```

### 数据转换中的行值替换
在数据转换时，可能需要根据业务规则对某些值进行替换。比如，将成绩等级进行转换：

```python
data_grades = {
    '成绩': ['A', 'B', 'C', 'A', 'B']
}
df_grades = pd.DataFrame(data_grades)

# 将成绩等级替换为对应的分数范围
grade_mapping = {
    'A': '90 - 100',
    'B': '80 - 89',
    'C': '70 - 79'
}
df_grades['成绩'] = df_grades['成绩'].replace(grade_mapping)
print(df_grades)
```

## 最佳实践

### 性能优化
当处理大规模数据时，性能是一个关键问题。尽量避免在循环中进行行值替换，因为循环的效率较低。可以使用向量化操作，例如上述的基于条件替换的方法，Pandas 会在内部进行优化，从而提高执行速度。

### 保持数据一致性
在进行行值替换时，要确保替换操作不会破坏数据的一致性。例如，在替换分类数据时，要确保新的值仍然属于该分类的有效范围。另外，在替换数值数据时，要注意数据类型的一致性，避免出现类型错误。
